FRR: Frequently recommended resources

Here is a roundup of resources we commonly recommend:

For those with little to no programming experience - or just general code kata sources for keeping sharp!

Python for Everybody <--- HIGHLY recommended

Automate the Boring Stuff with Python

Learn Python the Hard Way

Python Anywhere

Hour of Python

Learn Python

Codecademy - Codecademy offers a free beginner-friendly course in the basics of Python.

Free Code Camp provides an extensive list of Python tutorials.

Codesters

The Python Software Foundation Tutorial - Python's official tutorial

First-Timers Only(.com) - This website is devoted to getting beginners involved in the Open Source community.

After Hours Programming - This tutorial covers language fundamentals and some web development concepts.

Invent Your Own Computer Games with Python 4th Edition - This book teaches readers how to program in Python by making simple, classic games like Hangman and Tic Tac Toe.

General Resources

Google's Python Class - This class has a slight barrier to entry but covers great topics like Lists, Dictionaries, and Regular Expressions

RealPython - This site has tons of tutorials, some free and some paid. It covers a wide range of topics including language fundamentals, machine learning, testing, frameworks, and many more.

The Hitchhiker's Guide to Python - This is a comprehensive guide covering best practices and uses of Python.

StudyTonight - This website categorizes tutorials by difficulty so that you can either start with the basics or with a particular topic.
